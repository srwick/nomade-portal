# 🎓 Nomade Portal

Portal acadêmico desenvolvido como projeto universitário, com foco em oferecer uma interface moderna e intuitiva para acesso às informações dos alunos.

## 📖 Sobre o projeto

O **Nomade Portal** foi desenvolvido para simular um ambiente acadêmico, permitindo que alunos acessem informações importantes de forma rápida e organizada.

O objetivo do projeto foi aplicar conceitos de desenvolvimento web moderno utilizando React, TypeScript e Vite, seguindo boas práticas de organização e versionamento de código.

---

## ✨ Funcionalidades

- 📋 Visualização de informações acadêmicas
- 👤 Área do aluno
- 📚 Consulta de disciplinas
- 📅 Organização de informações em interface moderna
- 🎨 Interface responsiva
- ⚡ Navegação rápida utilizando React

## 🚀 Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/srwick/nomade-portal.git
```

### Entre na pasta

```bash
cd nomade-portal
```

### Instale as dependências

Caso utilize npm:

```bash
npm install
```

Ou, caso utilize Bun:

```bash
bun install
```

### Execute o projeto

Com npm:

```bash
npm run dev
```

Ou com Bun:

```bash
bun dev
```

O projeto estará disponível em:

```
http://localhost:5173
```
